# instagram

Basic Instagram App.

## Getting Started 

This project is a starting point for a Android application.


Features -
- Users can easily Post pictures.
- Search Users.
- Follow and unfollow.
- Like and Comment.
- User can save the posts.
- Notification, when user hits like and comments.
- Edit profile.
- Story (24hrs Time span).
